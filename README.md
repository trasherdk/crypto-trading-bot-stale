# crypto-trading-bot
Crypto trading bot for Binance

## Project Setup Guide

### Requirements
- `npm ^7.5.4.`
- `node ^15.10.0`
- `nodemon ^2.0.7`

### Build Setup
```bash
# copy .prod file
$ cp .env.prod .prod

# install dependencies
$ npm install

# serve with hot reload at localhost:3001
$ npm run dev
```

### Application entry point
> [index.js](https://github.com/driule/crypto-trading-bot/blob/main/index.js)
